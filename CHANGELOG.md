# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- **Documentation**
  - Comprehensive README.md in Russian with full project documentation
  - README_EN.md - English version for international users
  - INSTALL.md - Quick installation guide (bilingual)
  - CONTRIBUTING.md - Guide for contributors (bilingual)
  
- **Features**
  - Real-time video mosaic creation
  - SIFT and ORB feature detectors support
  - YOLOv8 object detection (people, vehicles, animals, etc.)
  - Automatic navigation map generation with pathfinding
  - A* algorithm for optimal route planning
  - GUI built with CustomTkinter
  - CLI mode for automation
  - Real-time mosaic visualization
  
- **Project Files**
  - requirements.txt with all necessary dependencies
  - Improved .gitignore with Python best practices
  
### Fixed
- Updated installation instructions for Linux/Ubuntu (tkinter requirement)

### Changed
- Enhanced .gitignore to exclude more temporary files and build artifacts
